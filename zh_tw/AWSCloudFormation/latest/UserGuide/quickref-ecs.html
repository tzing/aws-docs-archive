Amazon Elastic Container Service 範本程式碼片段

Amazon Elastic Container Service (Amazon ECS) 是一種容器管理服務，可以在 Amazon Elastic Compute Cloud (Amazon EC2) 執行個體叢集上輕鬆執行、停止及管理 Docker 容器。

下列範例範本會使用自動調整規模和和 Application Load Balancer，在 Amazon ECS 容器中部署 Web 應用程式。如需詳細資訊，請參閱《Amazon Elastic Container Service 開發人員指南》中的 Amazon ECS 入門。

重要

如需取得最新的 AMI ID，請參閱《Amazon Elastic Container Service 開發人員指南》中的 Amazon ECS 最佳化 AMI。
